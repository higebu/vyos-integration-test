# vyos-integration-test

VyOS integration test with Vagrant, KVM and Serverspec.

## Setup

```
git clone https://github.com/higebu/vyos-integration-test.git
cd vyos-integration-test
./setup.sh
```

## Run test

```
rake spec
```
